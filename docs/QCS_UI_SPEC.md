# QCS UI Spec (Phase 1)

## Color Tokens
- **Background**: `#07090f`
- **Surface**: `#0b1228`
- **Surface Soft**: `#0c142a`
- **Accent (Fuchsia)**: `#d946ef`
- **Accent 2 (Cyan)**: `#22d3ee`
- **Text Primary**: `#f8fafc`
- **Text Muted**: `#94a3b8`
- **Border**: `rgba(255,255,255,0.12)`

## Typography Scale
- **H1**: 32–40px (mobile), 44–56px (desktop), weight 800
- **H2**: 22–28px, weight 700
- **Body**: 15–16px, weight 400–500
- **Caption**: 11–12px, uppercase, letter-spacing 0.18–0.22em

## CTA System
- **Primary**: filled or gradient, fuchsia→purple→cyan, white text
- **Secondary**: outline with fuchsia stroke
- **Tertiary**: text or ghost, muted
- **Minimum height**: 44px

## Phase 1 Layout Upgrades
- **Events Hero**: gradient hero, prominent series focus, small info card
- **Filter Pills**: uppercase, pill style, fuchsia outline
- **Event Cards**: image-led or text-led cards with hover lift and border glow
- **Checkout CTA**: gradient primary button with subtle glow

## Accessibility
- Maintain 4.5:1 contrast for body text.
- Use scrims behind overlay text on imagery.
- Ensure focus outline is visible on interactive elements.
